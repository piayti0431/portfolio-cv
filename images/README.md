# Thư mục ảnh

Lưu 4 ảnh chụp màn hình số liệu vào đây, **đúng tên file** dưới đây (định dạng PNG hoặc JPG đều được, nhưng phải đúng tên `.png`):

| Tên file cần đặt      | Ảnh screenshot tương ứng                                   |
|-----------------------|------------------------------------------------------------|
| `quynhon-gsc.png`     | Google Search Console của **dulichquynhon.blog** (16,2K nhấp / 936K hiển thị) |
| `quynhon-ga4.png`     | Google Analytics 4 của **Blog Dulichquynhon** (34.055 phiên) |
| `vungtau-gsc.png`     | Google Search Console của **vungtau.travel** (459 nhấp / 15,5K) |
| `vungtau-ga4.png`     | Google Analytics 4 của **Vũng Tàu Travel** (944 phiên)     |

## Lưu ý
- Nếu ảnh của bạn là `.jpg`, hãy đổi phần đuôi `.png` thành `.jpg` **cả trong tên file lẫn trong `index.html`** (tìm `images/` để sửa), hoặc đơn giản nhất là đổi tên ảnh thành đuôi `.png`.
- Nên cắt gọn ảnh cho rõ phần biểu đồ + số liệu, dung lượng < 500KB để trang tải nhanh.
- (Tùy chọn) Thêm `avatar.jpg` nếu muốn dùng ảnh chân dung thật.
